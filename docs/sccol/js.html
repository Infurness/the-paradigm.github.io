@Commands

JS

Description:

Initiates JavaScript mode. All input will then be interpreted as JavaScript code. To exit - use the SCCOL command.

Example of usage:
JS **command submit**
Print("This text is being printed through JavaScript mode.");
Output:
This text is being printed through JavaScript mode.
Note:

JS should be written and executed as a single command. Switching to JavaScript mode mid-execution will not work. INTERNAL should be used instead.
