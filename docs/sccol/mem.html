@Commands

MEM (proc)

Description:

Displays the memory buffer, all declared procedures, or a single one's code on screen, if the procedure is specified.

Arguments:

PROCEDURE (proc) - the optionally specified procedure to display the code of.

Example of usage:
SET 0 SCP; PROC MAIN; SAY Secure. Contain. Protect.; END; MEM; MEM MAIN;
Output:
0: SCP MAIN; MAIN: SAY Secure. Contain. Protect.;
