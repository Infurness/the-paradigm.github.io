@Commands

FREE {num/start/proc} (end)

Description:

Frees specified process, a memory buffer block, or a range of them, previously set by the SET command - attempting to access the freed memory buffer blocks or procedures may result in an undefined value error.

Arguments:

INT {num/start/proc} - the specified memory block number to free, or a procedure. If the end value is specified, acts as the beginning of a range.

INT (end) - the optionally specified memory block range end. Specifying this will free all memory buffer blocks from the start to the end inclusive.

Example of usage:
SET 0 32; FREE 0; MEM;
