@Commands

SIN {num}

Description:

Sets the specified memory buffer block value in radians to its sine value. You may use %^D2R to convert degrees to radians.

Arguments:

INT {num} - the specified memory block number to get the value from and assign to.

Example of usage:
SET 0 (%^PI $/ 2); SIN 0; SAY %^0;
Output:
1
