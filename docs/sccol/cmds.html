@Commands

CMDS

Description:

Displays all available commands on screen.

Example of usage:
CMDS;
Output:
// (str) Declares a comment. ABS {num} Sets a memory buffer block to its absolute value. BG {col} Sets background color. CALL {proc} Calls a previously declared procedure. CHAR {num} (index) Converts memory buffer block value to character. CLR Clears the screen of all text. CMDS Displays all available commands. COS {num} Sets a memory buffer block to its cosine value (radians). END Ends current procedure declaration or exits the console. FRAC {num} Sets a memory buffer block value to its decimal remainder. FREE {num/start} (end) Frees a single or a range of memory buffer blocks. GET {num} Waits for user input to set its string to a memory buffer block. IF {(expr)} {cmd} If an expression returns true, executes a command. INFO (cmd) Displays client information or command information. INT {num} Converts memory buffer block value to integer. JS Initiates JavaScript mode. KEY {num} Waits for a keypress to set its value to a memory buffer block. MEM (proc) Displays the memory buffer or procedure code. PROC {name} Starts the declaration of a procedure. SAY {str} Displays a string of text. SCCOL Initiates SCCOL mode. SET {num} {val} Sets a memory buffer block to a value. SIN {num} Sets a memory buffer block to its sine value (radians). WAIT {ms} Pauses for an amount of milliseconds.
