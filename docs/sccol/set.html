@Commands

SET {num} {val}

Description:

Sets the specified memory buffer block to the specified value. The value can then be retrieved using %^**memory buffer block number**

Arguments:

INT {num} - the specified memory block number to assign to.

STR/REAL {val} - the specified value.

Example of usage:
SET 0 12; SET 1 (%^0 $* 2); SAY %^1;
Output:
24
