@Commands

INT {num}

Description:

Converts a floating point number or a character to an integer (Unicode value) in the specified memory buffer block.

Arguments:

INT {num} - the specified memory block number to get the value from and assign to.

Example of usage:
SET 0 %^PI; INT 0; SET 1 A; INT 1; MEM;
Output:
0: 3 1: 65
