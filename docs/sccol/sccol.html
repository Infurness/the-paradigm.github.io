@Commands

SCCOL

Description:

Initiates SCCOL mode. All input will then be interpreted as SCCOL commands. Is typically used to switch from another mode back to SCCOL.

Example of usage:
JS; **command submit**
Print("This text is being printed through JavaScript mode."); **command submit**
SCCOL; **command submit**
SAY This text is being printed through SCCOL mode.;
Output:
This text is being printed through JavaScript mode. This text is being printed through SCCOL mode.
