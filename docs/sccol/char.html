@Commands

CHAR {num} (index)

Description:

Converts a real number to a character (Unicode value), or gets a character at position (index) in a string in the specified memory buffer block.

Arguments:

INT {num} - the specified memory block number to get the value from and assign to.

INT (index) - the optionally specified character index to retrieve from a string (if a string is given).

Example of usage:
SET 0 83; CHAR 0; SET 1 SCP; CHAR 1 1; SAY %^0%^1P;
Output:
SCP
